PHP
Incepand cu acest curs, incep si PHP-ul

PHP este un limbaj de backend, asemanator cu JAVA. Acesta este un limbaj intrpretat si nu compilat. Java este un limbaj compilat.
Peste 95% din site-urile din internet sunt facute in PHP.
Incepeti cu primele 6 capitole:

Incepand cu acest curs o sa invatam si git.
Toate temele le veti comite in git.

Vom utiliza github. Acesta este un repozitoriu. Cu alte cuvinte, tot ce lucrati local o sa commiteti in github.
1) Va faceti un cont pe github.
2) Dupa ce ati intrat in cont, va duceti pe "Rpositories: si dati "New"
3) Dati numele ProiectITutor, acesta este public, dati ok
4) In directorul tema4, inainte sa incepeti dati, git clone ...... (copiati ce aveti la ssh in repo)
5) Incepeti sa lucrati local, dupa ce terminati faceti:
git add .
git commit -m "Trimitere tema"
git push origin dev
